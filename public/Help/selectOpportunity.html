What information is required to Create my engagement complexity assessment?

Complexity assessments are created in respect of opportunities, which are created and maintained on our Oracle CRM on Demand (OCOD) system.

All that is required to commence the engagement complexity assessment is:

  • The specific opportunity for which the complexity assessment is being completed; and
  • The next assessment gate - i.e. the next DCR or BCR gate in the business governance lifecycle.
  • You can optionally choose to also provide a "nickname" for your complexity assessment; something that might be easier to remember than the OCOD reference number, or the opportunity name.

You can type in the OCOD reference or the Opportunity name; alternatively you can find an opportunity via the Search for an opportunity button.

  • Note: the OCOD reference, the Opportunity name and the Next assessment gate are all mandatory fields; the Next button will only become active once these fields have been completed.

What if I don't know the OCOD reference number for the opportunity?

There are two ways that you can find an OCOD reference number for your opportunity:

  1. Both of the OCOD reference and the Opportunity name edit boxes have a "smart search" feature:
    If you click in one of these fields and start to type characters that you know are part of the reference/name then a drop-down list will appear to show you any opportunities on OCOD that contain the characters that you have typed. You can select an opportunity from this list.
    When you select an OCOD reference then the opportunity name is automatically populated for you; or

  2. You can press the Search for an opportunity button.
    This allows you to undertake a more extensive search.
    You can narrow down your search by selecting a specific Sector to search within.
    You can further narrow down by also selecting specific Account (within the Sector) to search within.
    This search feature will provide a table of valid opportunities from which you can make a selection.

What is the "Nickname"?

You can choose to give this complexity assessment a "nickname", so you have something that is easy to rememberfor future reference.

Adding a nickname is not mandatory, you may leave this field blank.

Please note that once the Nickname has been entered for a complexity assessment it cannot be changed/updated.

Selecting the Next assessment gate

You can select the Next assessment gate via a drop-down list; two fields are displayed - Gate and Gate Description.

When a new complexity assessment is being created the default value will be "Please select". You need to select the most appropriate gate from the drop-down list.

Please note that the Next assessment gate is mandatory - you have to select one of the gates from the drop-down list ("Please select" is not a valid selection).

What you have to do to update your engagement complexity assessment

When you are updating an existing engagement complexity assessment there is no need to change any of the existing information unless you need to.

The Next button will be active as you enter this page.

Please note that:

  1. You will not be able to edit either OCOD reference or the Opportunity name;
  2. If the Nickname is blank, then you can choose to give the complexity assessment a "nickname", so you have something that is easy to remember for future reference (adding a nickname is not mandatory, you may leave this field blank); if the complexity assessment already has a Nickname then you will not be able to change it; and
  3. With regards to the Next assessment gate you can change the existing selection, but when you change the next assessment gate a new complexity assessment shall be created for the opportunity, using the new next assessment gate.
