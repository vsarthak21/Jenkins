How do I complete this page?

Fourteen complexity factors

The engagement complexity assessment is comprised of fourteen complexity factors.

For each complexity factor you need to complete two actions:

  1. Enter the Complexity score for the complexity factor in question - this is achieved via a selection (of Very High; High; Moderate or Low) from a table which is accessed by pressing the Select button; and
  2. Enter the rationale that explains the principal reasons for selecting the Complexity score (per above) - this is done by typing text in to the Rationale for complexity score edit box.

Please note that:

Both of these fields are mandatory; and

The Rationale for complexity score edit box will accept up to 400 characters.

Four panels

The fourteen complexity factors are grouped into four main areas:

  • Client Factors;
  • Project Factors;
  • Commercial Factors; and
  • Capability Factors.

When you first enter this Engagement Complexity Assessment page you will see four ‘panels’ - labelled as Client Factors; Project Factors; Commercial Factors; and Capability Factors.
By default these panels are "collapsed".

Navigation between panels

At the end of each panel there is a right arrow symbol ">". When you click on this right arrow symbol the panel will open up and you will be able to enter or change the Complexity score and Rationale for complexity score for each of the complexity factors within the panel.

At this point there is an up arrow "^" at the end of the panel. When you click on this up arrow the panel shall collapse and you can open up a different panel.

At the bottom of the panel there are two buttons - Save and Next.

When you press the Next button the panel that you are in will collapse and the tool will open up so that you can enter or change the Complexity score and Rationale for complexity score for each of the complexity factors within that panel.

Completing the complexity assessment

At the end of each panel there is an "x/y" counter:

  • The x shows you how many of the complexity factors within the panel have been completed; and
  • The y shows you how many complexity factors there are in total within the panel.

You cannot proceed from this page until all fourteen complexity factors have been completed.

Once all fourteen complexity factors have been completed then press the Next button on the last (Capability Factors) panel to proceed to the Moderate & Submit page.

Please note that this Next button will remain disabled until all fourteen complexity factors have been completed.

Saving the complexity assessment

At the bottom of each of the four panels there are is a Save button.

You can press Save at any time to change the engagement complexity assessment that you are creating, or to save the changes that you have made to an existing complexity assessment that you are updating.

The Back button on the bottom of this page will let you navigate back to previous pages.

Important Security Considerations - Please Read

Whilst the Engagement Complexity Tool has been designed for "internal users" (i.e. users who are employed by Sopra Steria and have a valid AD-One user name and password) it is absolutely critical that data of a particularly sensitive or classified nature should not be included.

Where necessary, users are reminded that all data that is entered to create and update each complexity assessment is suitably obfuscated in order to protect sensitive information and to ensure that all client and data confidentiality obligations are strictly adhered to.

How do I use this page when I am Creating a new complexity assessment?

You need to complete the details required for each of the fourteen complexity factors, by navigating through each of the four panels.

Once all fourteen complexity factors have been completed then press the Next button on the last (Capability Factors) panel to proceed to the Moderate & Submit page.

You can partially complete your engagement complexity assessment, press Save and then navigate back to previous pages by using the Back button at the bottom of this Engagement Complexity Assessment page.

How do I use this page when I am Updating an existing complexity assessment?

You can make whatever changes are required to any of the fourteen complexity factors, by navigating through each of the four panels.

Once your changes have been made - ensuring that all fourteen complexity factors have been completed - then press the Next button on the last (Capability Factors) panel to proceed to the Moderate & Submit page.

You can come back to your engagement complexity assessment and make further changes at any point by pressing Save and then navigating back to previous pages by using the Back button at the bottom of this Engagement Complexity Assessment page.
