How do I use this page?

From this page you can specify a series of search criteria to help you find the engagement complexity assessment, or range of engagement complexity assessments that you need. They have been designed to help you 'narrow down' your search.
When you have specified your search criteria press the Next button. This will navigate you to the Search Results page, which provides you with a list from which you can select one or more existing engagement complexity assessments.
None of the search criteria are mandatory fields; pressing the Next button as soon as you enter this page is effectively a "find all" option.
In the same way, leaving one of the search criteria blank is effectively specifying "all".

The search criteria are:
  • Sector;
  • Account;
  • OCOD reference;
  • Opportunity name;
  • Data created - from; and
  • Date created - to.

Both the Sector and Account fields are drop-down lists from which you can make a selection. These will help you narrow down your search if you are trying to find a specific engagement complexity assessment to update - equally they will constrain the list of engagement complexity assessments that will appear in your engagement complexity report.
The Data created from and to dates will work in a comparable way. These are data fields that let you select dates from a calendar so you can specify a date range and only retrieve engagement complexity assessments that were submitted (i.e. either created or last updated) between that date range.
The OCOD reference and Opportunity name are on this page to help you find a specific engagement complexity assessment - either to update, of for which you want to download the engagement complexity assessment report. These are both edit boxes which have a "smart search" feature; they also have a link to a lookup feature (displayed as a magnifying glass icon).

How do I find one specific engagement complexity assessment?

You can specify search criteria that will narrow down the list of 'matching' engagement complexity assessments to a more manageable size and press the Next button to navigate to the Search Results page, from which you can select the engagement complexity assessment that you need.
If you have an understanding of the opportunity for which the engagement complexity assessment has been created, then it may be quicker to use either the OCOD reference or Opportunity name - either using the "smart search" feature or the lookup feature (displayed as a magnifying glass icon).

The Sector Lookup

There is a Sector drop-down list.
You can use this to narrow down the list of OCOD opportunities, to only display opportunities that 'belong' to the sector or JV that you have selected.
This drop-down list shows each of the possible values for sector within OCOD:
For example: ADS1-Defence Sector or ADS4-Secure (where the Super Sector is "Aerospace Defence & Security", as denoted by the prefix in the Sector Name of "ADS").
The drop-down list will display these values in alphabetical order (A to Z), sorted by Sector.

The Account Lookup

There is also an Account drop-down list.
This drop-down list will only be active when you have selected from the Sector drop-down list.
You can use this to narrow down the list of OCOD opportunities, to only display opportunities that 'belong' to the sector or JV - and the account - that you have selected.
This drop-down list shows all of the possible values for account within OCOD, for the specific sector that you have selected:
For example: MOD Southern Services (where the Sector is "ADS4-Secure")
The drop-down list will display these values in alphabetical order (A to Z).

OCOD reference

The OCOD reference edit box has a "smart search" feature:
If you click in this field and start to type characters that you know are part of the reference then a drop-down list will appear to show you any opportunities on OCOD that contain the characters that you have typed.
You can select an opportunity from this list.
When you select an OCOD reference then the Opportunity name is automatically populated for you.
Alternatively you can press the lookup button, displayed as a magnifying glass icon.
This allows you to undertake a more extensive search - you will be navigated to the Search Opportunities page, where:

  • You can narrow down your search by selecting a specific Sector to search within.
  • You can further narrow down by also selecting specific Account (within the Sector) to search within.
  • This search feature will provide a table of valid opportunities from which you can make a selection.

Opportunity name

The Opportunity name edit box has a "smart search" feature:
If you click in this field and start to type characters that you know are part of the opportunity name then a drop-down list will appear to show you any opportunities on OCOD that contain the characters that you have typed.
You can select an opportunity from this list.
When you select an Opportunity name then the OCOD reference is automatically populated for you.
Alternatively you can press the lookup button, displayed as a magnifying glass icon.
This allows you to undertake a more extensive search - you will be navigated to the Search Opportunities page, where:

  • You can narrow down your search by selecting a specific Sector to search within.
  • You can further narrow down by also selecting specific Account (within the Sector) to search within.
