What is a "Fast Track" complexity assessment?

A "fast track" engagement complexity assessment has been introduced to streamline the process for changes against an existing account-based portfolio.

An engagement will qualify for a "Fast Track" if all six specific criteria are met. These are:

  1. The engagement is a change to an existing account; and
  2. The commercial terms are well established, well understood, and do not feature penalties for late delivery; and
  3. The engagement is to be delivered by an existing Sopra Steria resource pool who understand the client environment; and
  4. We do not need to engage new subcontractors to deliver the engagement; and
  5. The cost of delivering the engagement does not exceed the "DCR delegated authority" thresholds (or equivalent) for the Sector; and
  6. The nature of the engagement does not significantly change or impact the existing Live Service.

When you are creating an engagement complexity assessment for the first time all six of these criteria are initially set to "No".

You will need to be able to faithfully answer "Yes" in all six cases to proceed with the "Fast Track" complexity assessment.

Where one or more of these criteria is not met - i.e. is set to "No" - then you will need to complete the 'full' engagement complexity assessment.

How do I use this page when I am Creating a new complexity assessment?

When you are creating a new engagement complexity assessment the first step is to determine whether or not the engagement qualifies for a "Fast Track" complexity assessment - there are six specific criteria defined that an engagement has to meet to qualify for a "Fast Track" complexity assessment.

This is done by asking you six specific questions.

When you are creating an engagement complexity assessment for the first time all six of these criteria are initially set to "No".

You will need to be able to faithfully answer "Yes" in all six cases to proceed with the "Fast Track" complexity assessment.

Where one or more of these criteria is not met - i.e. is set to "No" - then you will need to complete the 'full' engagement complexity assessment.

Please note that:

  1. If you initially create the engagement complexity assessment as a "Fast Track" complexity assessment you will be able to modify any of the six Fast Track criteria at a later point (which will then mean that you have to complete the 'full' complexity assessment for the engagement in question);
  2. If you answer "No" to one or more of the "Fast Track" criteria and create the 'full' engagement complexity assessment this can not be changed at a future point to a "Fast Track" complexity assessment.

How do I use this page when I am Updating an existing complexity assessment?

When you are updating an existing engagement complexity assessment the tool shall previously have determined whether or not the engagement meets the "Fast Track" criteria.

This page shall only appear when you press Next on the previous (Select Opportunity) page if the engagement complexity assessment that you are updating is a "Fast Track".

As you enter this page all six of these criteria are initially set to "Yes".

If you are still able to faithfully answer "Yes" in all six cases then press the Next button to proceed to the Moderate & Submit page.

If one or more of the "Fast Track" criteria is not met then you will need to set that to "No". Now press the Next button and you will to complete the 'full' engagement complexity assessment.
