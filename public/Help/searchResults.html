You have invoked help from the Search Results Page.

This page is used in two different scenarios. The functionality is a little different in either scenario:

Firstly where you need to find an existing engagement complexity assessment that you want to update, or for which you want to download the engagement complexity assessment report; and

Secondly it is also used to help you select a range of engagement complexity assessments when you are generating an engagement complexity report.

Please see below some information that will help you to use this page:

How do I use this page when I need to select a specific engagement complexity assessment to update?
How do I use this page when I need to select a specific engagement complexity so I can download the complexity assessment report?
How do I use this page when I need to select a range of engagement complexity assessments for my engagement complexity report?

How do I use this page when I need to select a specific engagement complexity assessment to update?

You will be presented with a list of existing engagement complexity assessments that match your search criteria.

You can press the Close button at any time to return to the Search Complexity Assessments Page and change your search criteria. The Close button is an "x " at the top of this page.

Use the scroll bar on the right hand side of the page to navigate through the list of engagement complexity assessments.

To select the engagement complexity assessment that you want to update then either double-click on the row or press the select icon on the end of the row (this is a white ">" within a black circle).

The engagement complexity assessment shall be retrieved and you will be navigated to the Select Opportunity & Gate page, from which you can work through the engagement complexity assessment and make any required modifications.

How do I use this page when I need to select a specific engagement complexity so I can download the complexity assessment report?

You will be presented with a list of existing engagement complexity assessments that match your search criteria.

You can press the Close button at any time to return to the Search Complexity Assessments Page and change your search criteria. The Close button is an "x" at the top of this page.

Use the scroll bar on the right hand side of the page to navigate through the list of engagement complexity assessments.

To select the engagement complexity assessment that you want to update then either double-click on the row or press the select icon on the end of the row (this is a white ">" within a black circle).

The engagement complexity assessment shall be retrieved and you will be navigated to the Select Opportunity & Gate page, from which you can navigate through the engagement complexity assessment pages to the Download your Complexity Assessment page.

How do I use this page when I need to select a range of engagement complexity assessments for my engagement complexity report?

You will be presented with a list of existing engagement complexity assessments that match your search criteria.

You can press the Close button at any time to return to the Search Complexity Assessments Page and change your search criteria. The Close button is an "x" at the top of this page.

Use the scroll bar on the right hand side of the page to navigate through the list of engagement complexity assessments.

You can select all of the engagement complexity assessments that are on the list by pressing a "select all" check box at the end of the table header row (to the right of the Next assessment gate column).

Alternatively you can select one or more individual engagement complexity assessments by clicking on the check box at the end of the row.

You will notice that there is a Continue with selection button at the bottom of this page.

This will show you the number of complexity assessments that you have currently selected for your report.

When you press the Continue with selection button your engagement complexity report will be generated and you shall be navigated to the Download your Report page.

Please note: If you select one engagement complexity assessment then you will be navigated to the Download your Complexity Assessment page so can download the engagement complexity assessment report, so that you can print, save and share the details and outcomes from the complexity assessment that you have selected.
