What is the Engagement Complexity Tool?

This CP-O02-TO-001 Engagement Complexity Assessment Tool is the leading risk indicator tool that is used to determine the appropriate level of surveillance and governance to ensure that we deliver in a predictable and profitable manner.

Completing the engagement complexity assessment allows us to understand objectively and consistently the levels of complexity - hence risk - inherent in the engagements that we sell and deliver.

There is a particular emphasis on completing the engagement complexity assessment at the earliest opportunity during the bid cycle. However, it should also be used on existing accounts where we are required to deliver new or additional commitments within an existing contract.

Typically the engagements that we undertake will comprise of one or more programmes or projects, which may or may not lead to the delivery of services.

This on-line tool has replaced the previous MS Excel version of the tool.

Please refer to CP-002-WI-001 Engagement Complexity Tool Work Instruction on iQ for guidance and/or use the help functions within the application.

Please also refer to the CP-O02-FR-001 Enterprise Risk Management Framework on iQ.

What does "Engagement" mean?

The term "engagement" is used to describe a contractual relationship that we (Sopra Steria; the Joint Ventures; or the Infrastructure Management Service Line) enter into with a client. This places price; scope; time; quality; and/or acceptance criteria obligations on us that - if not effectively managed - will result in some form of financial or reputational damage.

This guidance puts a particular emphasis on completing the assessment at the earliest opportunity during the bid cycle. However, it should also be used on existing accounts where we are required to deliver new or additional commitments within an existing contract.

Typically the engagements that we undertake will comprise of one or more programmes or projects that may or may not lead to the delivery of services.

About the Engagement Complexity Tool

What? Based upon the objective assessment of fourteen complexity factors, the Engagement complexity tool determines the overall complexity rating for the Engagement in question, ranging from Very High to High, Moderate or Low complexity.

Why? The Engagement complexity tool enables a robust risk assessment. It also objectively informs key business decisions regarding the most appropriate: level of surveillance; Engagement assurance that needs to be applied; and proportionate adoption of the business management system.

Who? The completion of the Engagement complexity assessment must be a collaborative undertaking by a small team of individuals who collectively have an ability to assess all fourteen complexity factors for each Engagement. As a minimum this should include solution; commercial and delivery contributions - and be owned by the Surveillance Director.

When? The Engagement complexity tool should be applied at the first opportunity during the BCR0 stage, but must have been completed prior to the BCR0c. The Engagement complexity assessment needs to be reviewed, and refined where appropriate, at each of the DCR and BCR gates thereafter.

Using the Engagement Complexity Tool

What does it do?

The engagement complexity tool allows you to perform one of the following functions:

  1. Create a new engagement complexity assessment;
  2. Update an existing engagement complexity assessment; and
  3. Generate a report containing engagement complexity reports generated via the complexity tool.
There is also a Trial Feature which allows you to use the complexity tool without having to actually complete and save an engagement complexity assessment.

You can use this option via the Try the complexity assessment tool button.

What is an engagement complexity assessment?

  • Engagement complexity assessments are created via the tool for specific opportunities;
  • These opportunities are created and maintained on our Oracle CRM on Demand (OCOD) system;
  • These opportunities can be either 'pre-sales' or 'in delivery' (i.e. contracted);
  • An engagement complexity assessment can be performed on a number of occasions for the same opportunity - the complexity of an engagement shall change over time as it progresses through our business governance and lifecycle - i.e. BCR and DCR gates;
  • The index key/unique identifier for each complexity assessment is the OCOD Reference + the Next Assessment Gate.

How do you complete a complexity assessment?

The process is straight forward and requires the following steps to be followed:

  1. Select Opportunity & Gate: Select the specific OCOD opportunity for which the complexity needs to be established and select the next assessment gate i.e. the next DCR or BCR gate in the business governance lifecycle;
  2. Fast Track?: determine whether the engagement qualifies as a "Fast Track" complexity assessment;
  3. Complete the complexity assessment: by completing each of the complexity factors;
  4. Moderate and submit: determine whether the complexity assessment needs to be moderated and submit the complexity assessment; and
  5. Report: download/save/print the complexity assessment report.

Important Security Considerations - Please Read

Whilst the Engagement Complexity Tool has been designed for "internal users" (i.e. users who are employed by Sopra Steria and have a valid AD-One user name and password) it is absolutely critical that data of a particularly sensitive or classified nature should not be included.

Where necessary, users are reminded that all data that is entered to create and update each complexity assessment is suitably obfuscated in order to protect sensitive information and to ensure that all client and data confidentiality obligations are strictly adhered to.

My recent complexity assessments

On the Home Page you will have a table called "My recent complexity assessments".

This provides you with a short-cut to retrieve (and update if required) those complexity assessments that you have created or updated in recent times.

Please note that only complexity assessments that you have created or updated in the last 3 months shall be shown; and a maximum of ten of your most recent complexity assessments shall be displayed

What reports are available?

Whenever you create or update a complexity assessment the final stage, after you submit the engagement complexity assessment, is the option you have to download and/or print a specific engagement complexity assessment report.

There is also an option to generate a report (in MS Excel format) for a range of engagement complexity assessments that have been created - you can select this option via the Extract reports button on this Home page.

I want to download or print the engagement complexity report for a specific engagement:

You can do this in one of two ways:

  1. Select Update complexity assessment; select the specific engagement and follow the pages through until you press the Submit button on the Moderate & Submit page. At this point you will be taken to the Download your Complexity Assessment page.
  2. Select Extract reports; from the Search Complexity Assessments page you can choose to select one specific engagement.
    At this point you will be taken to the Download your Complexity Assessment page.

I want to create and download or print for a range of engagement complexity assessments that have been created:

You can do this via the Extract reports button on this Home page.

What are the help features that are available?

A help page like this one has been created for each of the pages within the engagement complexity tool. These are always activated by clicking the "Help with this page" link.

Hover on Help refers to a short, explanatory message that is displayed to help clarify for you what is required when you locate the cursor 'above' a particular field/button/feature.

The message will disappear as soon as you move the cursor away from the field/button/feature.

What are the user navigation features that are available?

You will navigate through each of the pages using Next and Back buttons:

  • The Next button will be active when all required fields on the page have been completed and will take you to the page that allows you to complete the next step on the complexity assessment process;
  • The Back button will allow you to return to the previous page.

When you have completed and submitted your complexity assessment the Home button will return you to this Home page.

When you want to exit from the engagement complexity tool you can Logout by using the "Log out" link.

  • Note: Logging out from the application does not close the browser tab, you will need to do this manually. The tab will otherwise remain open on the Login page.
