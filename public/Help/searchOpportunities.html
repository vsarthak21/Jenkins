You have invoked help from the Search Opportunities page.

This page is used to help you find the OCOD opportunity for which you need to create the engagement complexity assessment; it is also used to help you find an opportunity when you are generating an engagement complexity report.

From this page you can select one, specific OCOD opportunity from a list.

Please see below some information that will help you to use this page:

The Sector Lookup

There is a Sector drop-down list.

You can use this to narrow down the list of OCOD opportunities, to only display opportunities that 'belong' to the sector or JV that you have selected.

This drop-down list shows each of the possible values for sector within OCOD:

For example: ADS1-Defence Sector or ADS4-Secure (where the Super Sector is "Aerospace Defence & Security", as denoted by the prefix in the Sector Name of "ADS").

The drop-down list will display these values in alphabetical order (A to Z), sorted by Sector.

The Account Lookup

There is also an Account drop-down list.

This drop-down list will only be active when you have selected from the Sector drop-down list.

You can use this to narrow down the list of OCOD opportunities, to only display opportunities that 'belong' to the sector or JV - and the account - that you have selected.

This drop-down list shows all of the possible values for account within OCOD, for the specific sector that you have selected:

For example: MOD Southern Services (where the Sector is "ADS4-Secure")

The drop-down list will display these values in alphabetical order (A to Z).

The Search Button

If you select a sector from the sector drop-down list, or change the values in either the sector or account drop-down lists then pressing the Search button shall re-fresh the list of opportunities within the table below the drop-down lists, so that you are better able to find the opportunity that you need.

How do I use this page when I am creating a new complexity assessment?

This page will allow you to select a specific opportunity from a list.

Using the sector and account drop-down lists - then pressing the Search button - will narrow down the list of opportunities, making it easier for you to find the opportunity that you need.

The opportunity that you select will then be populated within the Select Opportunity & Gate page - the OCOD reference and the Opportunity name fields will be populated with the correct values.

You will have the option to add a Nickname.

You will then select the Next assessment gate and continue with the creation of your engagement complexity assessment.

How do I use this page when I am creating a complexity assessment report?

This page will allow you to select a specific opportunity from a list.

Using the sector and account drop-down lists - then pressing the Search button - will narrow down the list of opportunities, making it easier for you to find the opportunity that you need.

The opportunity that you select will then be populated within the Search Complexity Assessments page - the OCOD reference and the Opportunity name fields will be populated with the correct values.

You can then enter any further report criteria and press the Next button to generate your complexity assessment report.
